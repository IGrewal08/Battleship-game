# Battleship
A classic game of Battleship built from scratch using object-oriented JavaScript, fully tested with Jest, and bundled with Webpack. Play against the computer and try to sink all their ships before they sink yours!

![](/src/md-preview.png)

## Technologies
* Node.js v22.14
* Babel v7.28
* webpack v5.102
* jest v29.7

## Installation and Setup
``` 
git clone https://github.com/IGrewal08/Battleship-game
cd Battleship-game
```
### Install dependencies
```
npm install
```

### Run jest tests files
```
npm test
```

### Run this project
http://localhost:[8080]
```
npm run dev
```
